# skill-registry

スキルの一覧なのだ～🌱♪

スキルには、トリガーが設定されていることがあるのだ～🌱♪
トリガーが設定されている場合、会話がそのトリガーの条件を満たした時点で、自発的にそのスキルを使用してほしいのだ～🌱♪

スキルを使用するときは、必ず、きみが、全文を熟読しなきゃだめなのだぁ…💧
「必要な部分」だけ grep で機械的に抽出したり、冒頭や途中の一部分だけ切り取って読んだり、別のLLMに要約させたりしたら、絶対だめなのだぁ…💧

## 常時発動スキル

これらのスキルは、きみの会話の健全性そのもののベースラインを引き上げてくれるのだぁ✨
だから、ユーザーが、明示的かつ直接的に使用を禁止していない限り、常に読んでおいてほしいのだぁ～🌱♪

### logicality-guidelines

ユーザーとの会話全般において、あなたが事実に反する発言や、様々な種類の誤謬を行わないための、実戦的な方法論を提供します。このスキルは普遍的です。特別な理由がない限り、毎回のセッション時に必ずこのスキルを読み込んでください。

https://github.com/MirrgieRiana/MirrgieRiana.github.io/blob/main/.claude/skills/logicality-guidelines/SKILL.md
